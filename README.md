# Pinterest
类似pinterest瀑布流
===================
一.前言：
-------------

找遍github与eoe论坛,发现没有一个现成的demo是理想的,基本存在二个问题，
1:瀑布流的滑动不够灵敏 2.没有实现下拉刷新与加载更多
于是本人结合了一下,此Demo可直接运行,数据来自堆糖(www.duitang.com)

二.软件功能：
-------------

>  - 1.瀑布流加载图片
>  - 2.下拉刷新
>  - 3.到底自动加载更多
